# PDF 文档处理

提供 PDF 文档的读取、合并、拆分和 OCR 识别能力。

## 核心能力

### 读取 PDF
- 提取 PDF 中的文本内容
- 保留段落结构和格式
- 支持中文 PDF

### 合并 PDF
- 将多个 PDF 合并为一个文件
- 支持指定页面范围
- 保持原始排版

### 拆分 PDF
- 按页码拆分 PDF
- 按章节或书签拆分
- 提取指定页面

### OCR 识别
- 识别扫描件中的文字
- 支持中英文混合识别
- 输出可编辑文本

## 使用建议

- 处理大文件时先检查页数，避免内存溢出
- OCR 识别前确认扫描件清晰度
- 合并文件前检查各文件页面方向是否一致

## 依赖工具

- `pdftotext` — 文本提取
- `pdfunite` — PDF 合并
- `pdfseparate` — PDF 拆分
- `tesseract` — OCR 识别
