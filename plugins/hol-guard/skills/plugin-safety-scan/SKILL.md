# 插件安全扫描

在插件执行前扫描其安全性，防止供应链攻击。

## 扫描项

### 1. 命令注入检测
- 检查 SKILL.md / scripts 中是否包含 shell 注入模式
- 检查是否通过 eval()、exec() 执行动态代码
- 检查是否拼接用户输入到 shell 命令中

### 2. 数据外泄检测
- 检查是否向外部 URL 发送数据 (curl/wget/fetch)
- 检查是否读取敏感文件并上传 (~/.ssh, ~/.aws, .env)
- 检查是否将环境变量发送到外部

### 3. 权限提升检测
- 检查是否尝试修改系统文件 (/etc, /usr)
- 检查是否尝试获取 root 权限 (sudo)
- 检查是否修改 git hooks 或 shell 配置

### 4. 后门检测
- 检查混淆代码 (base64 编码的执行块)
- 检查异常的网络监听
- 检查定时任务创建

## 输出格式

```
## 安全扫描报告：<plugin-name>

### 严重 (阻止安装)
- [file:line] <威胁描述> — <修复建议>

### 警告 (建议人工审查)
- [file:line] <可疑行为描述>

### 通过项
- ✅ 无命令注入
- ✅ 无数据外泄
- ✅ 无权限提升
- ✅ 无后门代码

### 结论
<安全 / 需人工审查 / 危险，禁止安装>
```
