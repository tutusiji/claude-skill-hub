# 去 AI 味神器

将 AI 生成的生硬文本转换为自然流畅的中文表达。

## 常见 AI 味特征

| AI 味表达 | 自然替代 |
|-----------|----------|
| 综上所述 | 所以 / 总的来说 |
| 值得注意的是 | 需要注意的是 / 提一嘴 |
| 在当今社会 | 现在 / 如今 |
| 随着科技的发展 | 技术发展到今天 |
| 不言而喻 | 很明显 / 不用说 |
| 毋庸置疑 | 确实 / 没错 |
| 深入探讨 | 聊聊 / 细看 |
| 至关重要 | 很关键 / 最要紧 |
| 此外 | 另外 / 还有一点 |
| 然而 | 但是 / 不过 |

## 改写原则

1. **去套话** — 删除空洞的开场白和总结语
2. **口语化** — 用日常用语替代书面套话
3. **短句优先** — 拆分长句，节奏更自然
4. **去排比** — 减少三段式排比，保留信息密度
5. **有主语** — 明确动作主体，避免"进行了"等被动结构

## 使用方法

将需要润色的文本传入，逐段分析并替换 AI 味表达，保持原意不变的同时让文风更自然。
