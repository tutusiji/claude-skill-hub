# OWASP Top 10 漏洞审计

按 OWASP Top 10 (2021) 逐项扫描代码安全漏洞。

## 审计清单

### A01 — 访问控制失效
- 越权访问：接口未校验资源归属
- 路径遍历：`../` 未过滤
- CORS 配置过宽：`Access-Control-Allow-Origin: *`
- 缺少权限分层：普通用户能访问管理接口
- JWT 未校验签名或过期时间

### A02 — 加密失效
- 明文传输：HTTP 接口未强制 HTTPS
- 弱加密算法：MD5、SHA1 用于密码存储
- 硬编码密钥：AES/RSA 密钥写在源码里
- 随机数不安全：用 `Math.random()` 生成 token

### A03 — 注入
- SQL 注入：字符串拼接 SQL，未用参数化查询
- 命令注入：`exec(userInput)` 未过滤
- LDAP/XPath/NoSQL 注入
- 模板注入：用户输入直接进模板引擎

### A04 — 不安全设计
- 缺少速率限制：登录/注册接口可暴力刷
- 敏感操作无二次确认
- 业务逻辑漏洞：负数金额、并发重复提交

### A05 — 安全配置错误
- 调试模式开启：`DEBUG=True` 上生产
- 默认凭证未改：admin/admin
- 错误信息泄露堆栈
- 目录列表开启

### A06 — 易受攻击的组件
- 依赖版本有已知 CVE
- 引入了废弃/无人维护的包
- 见 dependency-scan 技能详细排查

### A07 — 认证与身份失效
- 密码强度未校验
- 会话固定：登录后不刷新 session ID
- 会话不过期：永久 token
- 多因素认证缺失

### A08 — 软件与数据完整性
- 反序列化未校验：`pickle.loads(user_data)`
- CI/CD 管道未校验构建产物
- CDN 资源无 SRI 校验

### A09 — 日志与监控失效
- 安全事件无日志
- 日志包含敏感信息（密码、token）
- 无异常告警机制

### A10 — 服务端请求伪造 (SSRF)
- URL 未做内网地址过滤
- 图片代理/预览功能可访问内网
- DNS rebinding 风险
