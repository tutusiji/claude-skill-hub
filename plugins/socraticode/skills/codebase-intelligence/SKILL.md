# 代码库智能搜索

使用混合语义搜索引擎定位代码，而非简单的文本匹配。

## 适用场景

- "这个功能的实现在哪里？" — 语义搜索找到相关代码
- "谁调用了这个函数？" — 符号级调用关系追踪
- "这个接口的所有实现" — 接口/抽象类的所有实现类
- "类似的代码模式" — 找到重复或相似结构的代码

## 工作流程

1. 解析用户意图：是在找定义、调用、引用还是相似模式
2. 执行混合搜索：语义向量匹配 + 精确符号匹配
3. 按相关性排序结果，标注匹配原因
4. 提供代码上下文（前后 10 行）

## 输出格式

```
## 搜索结果："<query>"

### 最佳匹配
[file:line] 匹配原因: <semantic/exact/symbol>
  <代码片段>

### 其他匹配
- [file:line] 匹配原因
```
